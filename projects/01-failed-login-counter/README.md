# Project 01: Failed Login Counter

## Overview
This project is a small Python script that reads sample authentication log data and counts failed login attempts.

## Objective
To build a simple and readable script that supports beginner log analysis and helps develop basic Python skills for cybersecurity.

## Why this project matters
Failed login activity can be normal, but repeated failures may also indicate:
- password guessing
- brute-force attempts
- account misuse
- misconfigured automated processes

A small script that counts these events is a good beginner exercise in both Python and security analysis.

## Planned Input
A text file containing sample log lines.

## Planned Output
A summary showing:
- number of failed login attempts
- usernames involved
- source IP addresses, when available

## Skills Practiced
- reading files in Python
- searching for patterns in text
- counting repeated events
- producing a simple report

## Next Step
Create the first version of the script and test it against sample log data.
